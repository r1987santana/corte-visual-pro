# Turquesa Restaurante OS - Concepto v1

Fecha: 2026-06-21

## Objetivo

Crear el software operativo de Turquesa Restaurante como un command center real para gerencia, caja, salon, cocina y operaciones.

## Superficie inicial

- POS rapido.
- Mapa de mesas.
- Comandas activas.
- Cocina / KDS.
- Reservas.
- Inventario critico.
- Cierre de caja.
- Clientes capturados por el portal WiFi.
- Propina 10% conectada con el modelo RRHH existente.
